# 浏览器渲染方式

浏览器对CSS阻塞渲染有两种处理方式，要么等css解析完一起渲染，chrome就是这么做的，但是会造成白屏；要么先把无样式的dom渲染出来等css解析好了再渲染一次，Firefox就是这么做的，但是会造成无样式内容闪烁

# 消除空隙

给父元素设置 font-size: 0 来消除 inline-block 元素之间的空隙。
